# Recipe Combiner App

An app that takes multiple recipe (ideally for the same thing) urls and uses AI to determine a similarly porportioned final recipe.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
## To run a production veriom

Run `npm run build` then `npm run start`.
